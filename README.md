# PlacePrep 🚀

PlacePrep is a comprehensive placement preparation platform designed to help students prepare for internships and campus placements efficiently. The platform provides curated learning resources, aptitude practice, coding preparation, interview guidance, and progress tracking in a single place.

## 📌 Overview

The goal of PlacePrep is to bridge the gap between academic learning and industry hiring requirements by providing students with structured preparation paths for:

- Aptitude & Reasoning
- Technical Interview Preparation
- Coding Practice
- Resume Building
- Company-Specific Preparation
- Placement Resources

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/valiantvedansh-rgb/placeprep_rgb.git
```

### Navigate to the Project

```bash
cd placeprep_rgb
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory and add:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
